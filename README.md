# 📊 Financial Performance Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze financial performance using advanced Time Intelligence techniques.

## 🚀 Key Features
- Revenue, Profit, and COGS Analysis
- Time Intelligence (YTD, MTD, QTD, YoY, MoM)
- Trend Analysis over time
- Dynamic visualizations
- Clean and professional UI design

## 📈 Insights Provided
- Year-to-date revenue growth
- Monthly performance trends
- Comparison with previous periods
- Segment and country-level performance

## 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Time Intelligence Functions
